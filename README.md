# multilabel-BERTs
BERTs for Multilabel Classification based on simpletransformers (and Huggingface transformers)

# Requirements
* Python 3.6.*
* Pytorch 1.0.0+
* simpletransformers
* Pandas
* Huggingface Transformers

Run bibsonomy_multilabel_bert.py to see results (currently the program needs fixing).
